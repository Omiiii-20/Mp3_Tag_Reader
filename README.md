MP3 Metadata Editor
A command-line utility written in C for viewing and editing ID3 metadata tags in MP3 files. This tool allows you to read, display, and modify common MP3 tags such as title, artist, album, year, genre, and comments.
🎵 Features

View MP3 Tags: Display all ID3 metadata tags from MP3 files
Edit MP3 Tags: Modify existing metadata tags with new values
ID3v2 Support: Works with ID3v2 tag format
Command-Line Interface: Simple and efficient terminal-based operation
Cross-Platform: Works on Linux, macOS, and Windows

Supported Tags
The program reads and modifies the following ID3 tags:

Title (TIT2)
Artist (TPE1)
Album (TALB)
Year (TYER)
Genre (TCON)
Comment (COMM)

Prerequisites

C Compiler: GCC or any C99-compatible compiler
Standard C Libraries: stdio.h, string.h
Custom Header: file.h
